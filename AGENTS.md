## Plannotator Skills (Global)

Los siguientes skills de Plannotator están disponibles globalmente y pueden invocarse con la herramienta `skill`:

- `plannotator-annotate` - Abrir UI de anotación para archivos markdown, HTML, URLs o carpetas
- `plannotator-compound` - Analizar archivos de plan de Plannotator para extraer patrones de denial y generar dashboard HTML
- `plannotator-last` - Anotar el último mensaje del asistente renderizado
- `plannotator-review` - Abrir UI de code review basada en navegador
- `plannotator-setup-goal` - Crear paquetes de setup de goals para trabajo longo con Plannotator

Para usar un skill, usa la herramienta `skill` con el nombre correspondiente cuando la tarea lo requiera.
